# Imagen de Comprobante de Pago

Coloca aquí la imagen estática que se enviará al canal público cuando se confirme un pago.

## Nombre del archivo
La imagen debe llamarse: `receipt-success.png`

## Ubicación
`backend/assets/receipt-success.png`

## Formato
- Formato: PNG
- La imagen debe ser la que muestra "¡Pago Exitoso! LA OPERACIÓN FUE PROCESADA" con el checkmark verde

## Nota
Si la imagen no existe, el sistema usará una imagen de fallback automáticamente.
